## 2. Как запустить локально

### Предварительный просмотр (без Telegram)
Открыть файл `preview.html` в браузере — покажет приложение в рамке iPhone с автоперезагрузкой каждые 2 секунды.

### Полноценный тест (только в Telegram)
1. Открыть Telegram на **телефоне** (Android или iOS)
2. Перейти по ссылке: `https://t.me/anna_beauty_nail_bot?start=from_app`
3. Нажать кнопку **"Записаться 💅"** внизу чата

> ⚠️ Telegram Desktop (компьютер) часто кеширует старые версии. Для надёжного теста использовать телефон.

- На экране успеха: кнопка "📅 Мои записи"

> ⚠️ Напоминание "придёт в Telegram за 24 часа и за 2 часа" — это UI-текст, реальных уведомлений нет (нет бэкенда)

## 4. Что хранится в localStorage

| Ключ | Значение | Описание |
|---|---|---|
| `onboarding_done` | `"1"` | Онбординг уже был показан |
| `offer_shown` | `"1"` | Оффер 15% уже был показан |
| `bookings` | JSON массив | Активные записи клиента |
| `historyBookings` | JSON массив | История отменённых записей |

## 5. Известные ограничения

- **Напоминания не работают** — нет бэкенда, текст "напоминание придёт" — UI-заглушка
- **Данные клиента не сохраняются на сервере** — всё только в localStorage на устройстве
- **Расписание захардкожено** — `busySlots` в `data.js`, реального календаря нет
- **Имя пользователя** в онбординге отображается только в настоящем Telegram (не в браузере)
- **Тёмная тема** на Telegram Desktop может работать некорректно из-за кеша
- **Кеш Telegram Desktop** — при проблемах: полностью закрыть и открыть Telegram

## 6. Деплой

Изменения в `tg-app/` → push в `main` → Vercel автоматически деплоит.

При изменении кода обязательно:
1. Поднять `?v=N` во всех трёх тегах в `index.html` (style.css, data.js, app.js)
2. Обновить URL кнопки бота через Telegram API (setChatMenuButton)
